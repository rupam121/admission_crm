# Admission CRM System (MERN)

## Overview
A simple Admission Management system to manage programs, applicants, seat allocation, and admission confirmation.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- User Authentication (JWT)
- Program & Quota Setup
- Applicant Management
- Seat Allocation (No Overbooking)
- Admission Confirmation
- Dashboard
- Apply
- fee: Paid / Pending

## Setup

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev

## Environment Variables
Create .env in backend:
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
PORT=5000
